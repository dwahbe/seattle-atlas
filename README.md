# Seattle Atlas

An interactive map for exploring Seattle's zoning and transit data.

**Live:** [atlas-two-zeta.vercel.app](https://atlas-two-zeta.vercel.app)

## Data Sources

- **Zoning:** City of Seattle Open Data Portal
- **Transit:** King County Metro GTFS

## Built With

- Next.js + TypeScript
- Mapbox GL JS
- Tailwind CSS
- Vercel
